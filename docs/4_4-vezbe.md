# Vežba 👷
* Napravi tekstualni fajl koristeći `vim` editor
* Upiši neki tekst
* Probaj `undo` komandu
* Dopiši još teksta
* Sačuvaj fajl
* Prikaži tekst upotrebom `cat` komande
* Napravi novi direktorijum
* Kopiraj kreirani fajl u nov direktorijum

# Dodatno 🔍
